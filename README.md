### iTerm2などのterminalにsolarizedを入れてください
[https://ethanschoonover.com/solarized/](https://ethanschoonover.com/solarized/)

### 必要ツールのインストール
- `gem install solargraph`
- `brew install cask`
- `brew install multimarkdown`

### 初期化コマンド
- `cd ~/.emacs.d`
- `cask install`

### パッケージのインストール方法
- `M-x pallet-install`
- `M-x pallet-update`
